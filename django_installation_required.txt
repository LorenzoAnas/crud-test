# Python Packages Installation

The following Python packages were installed for the Django project:

1. **django-cors-headers**: This package is used to handle the server headers required for Cross-Origin Resource Sharing (CORS) in a Django application.

    Installation command:
    ```bash
    pip install django-cors-headers
    ```

2. **psycopg2-binary**: This package is a PostgreSQL adapter for Python. It is used by Django to provide support for PostgreSQL database.

    Installation command:
    ```bash
    pip install psycopg2-binary
    ```

3. **djangorestframework**: This package is a powerful and flexible toolkit for building Web APIs in Django.

    Installation command:
    ```bash
    pip install djangorestframework
    ```
